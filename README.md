# Basketball Coach AI

Expo + React Native + TypeScript app for basketball practice coaching.

## Run In Expo Go

```bash
npm install
npm start
```

`npm start` now uses `expo start --tunnel`, so you can open the project from `Expo Go` on your phone even when the phone is not on the same local network.

Useful commands:

```bash
npm run start:phone
npm run start:lan
npm run start:local
npm run android
npm run ios
npm run web
npm run doctor
```
